## データ型

- オブジェクト
  - 1 次元: シリーズ `pandas.Series`
  - 2 次元: データフレーム `panas.DataFrame`

DataFrame の列や Series は、以下のデータ型の値を取る

| dtype         | 対応する Python のデータ型 |
| ------------- | -------------------------- |
| object        | str                        |
| int64         | int                        |
| float64       | float                      |
| bool          | bool                       |
| datetime64    |                            |
| timedelta[ns] |                            |
| category      |                            |
