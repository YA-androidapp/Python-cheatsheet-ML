### Dataframe の概要を確認

#### 行を絞る

```py
import numpy as np
import pandas as pd

arr = np.arange(100).reshape(10, 10)
idx = pd.Index(['r{}'.format(x+1) for x in range(10)], name = 'index')
col = pd.Index(['c{}'.format(x+1) for x in range(10)], name= 'column')
df = pd.DataFrame(arr, index=idx, columns=col) # 2次元: データフレーム

df.head() # 先頭5行
df.tail() # 末尾5行
```
