#### 行名・列名、行数・列数・要素数

```py
import numpy as np
import pandas as pd

df = pd.DataFrame(np.arange(50).reshape(5, 10), index=pd.Index(['r{}'.format(x+1) for x in range(5)], name = 'index'), columns=pd.Index(['c{}'.format(x+1) for x in range(10)], name= 'column'))

df.index # 行名
df.columns # 列名
df.values # 値
```

